InstallationÂ¶

Download the latest release at: https://github.com/ndevenish/Blender_ioEDM/releases/latest

Launch Blender and open up the User Preferences from the File menu. Switch to the Addons tab and then click “Install From File...”

Navigate to where you downloaded Blender_io_EDM.zip and select the file. Blender will install io_EDM to a standard location, then show the newly installed addon in the list. Simple then check the ‘Enabled’ box to finish installing:

You can also click ‘Save User Settings’ to keep the addon loaded next time you start blender - If you do this, it will not interfere with any other uses of Blender, but will leave the edm-specific user interfaces in place until you disable the addon.
